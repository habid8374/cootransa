# SISTEMA DE CONTROL DE ASISTENCIA BIOMÉTRICO
### Propuesta de desarrollo — COOTRANSA Ltda.

**Preparado para:** Cooperativa de Transportadores de Sabanalarga (COOTRANSA Ltda.)
**Fecha:** Junio 2026
**Modalidad:** Proyecto independiente

---

## 1. El problema actual

El registro de asistencia del personal mediante planillas físicas o métodos manuales genera:

- Pérdida de tiempo en consolidación para nómina
- Posibilidad de errores y registros inexactos
- Dificultad para auditar horas reales trabajadas
- Consumo de papel y archivo físico

---

## 2. La solución propuesta

Un sistema digital de control de asistencia mediante **reconocimiento facial**, que registra automáticamente la entrada y salida de cada empleado desde una tablet o computador ubicado en el punto de acceso.

### ¿Cómo funciona?

```
Empleado se ubica frente a la cámara
        ↓
El sistema reconoce su rostro automáticamente
        ↓
Registra hora exacta de entrada o salida
        ↓
Queda almacenado en base de datos segura
        ↓
La administración genera reportes para nómina
```

---

## 3. Funcionalidades incluidas

| Funcionalidad | Descripción |
|---|---|
| **Registro facial** | Marcación de entrada/salida con la cámara, sin contacto |
| **Panel administrativo** | Alta de empleados, fotos de referencia, horarios |
| **Reportes de asistencia** | Por empleado, por fecha, por rango — exportables a Excel |
| **Cálculo de horas** | Horas trabajadas, llegadas tarde, ausencias |
| **Registro histórico** | Cada marcación queda registrada con fecha y hora exacta |
| **Acceso seguro** | Solo personal autorizado consulta los reportes |

---

## 4. Tecnología

- Reconocimiento facial mediante inteligencia artificial
- Funciona en tablet, celular o computador con cámara
- Datos almacenados de forma segura en la nube
- **No requiere hardware especializado** (solo una tablet o cámara web)

---

## 5. Beneficios para COOTRANSA

✓ Elimina las planillas de papel — apoyo al medio ambiente
✓ Nómina más rápida y precisa
✓ Control real de horarios y puntualidad
✓ Información disponible al instante para auditoría
✓ Imagen moderna y profesional de la cooperativa
